# Car Rental System (Java Console Project)

A simple console-based **Car Rental System** built using Core Java.  
This project is designed for beginners to understand OOP concepts and create an interactive menu-driven application.

---

##  Features

- User Login (Username + Password)
- Choose Car Type:
  - Mini
  - Sedan
  - SUV
- Different fare per kilometer for each car type
- Enter distance to calculate total fare
- Display final bill
- Simple & beginner-friendly Java code

---

### ✔ Object-Oriented Programming

##  How to Run

1. Download or clone this repository  
2. Open the folder in any IDE like IntelliJ, Eclipse, VS Code, NetBeans  
3. Run the file:  
   ```
   Main.java
   ```
4. Follow the menu options in the console
